  A progressive Hacker News client built with React + Vite + TypeScript
</p>

---

:zap: **Fast:** Service Worker App Shell + Dynamic Content model to achieve faster load times with and without a network.

:iphone: **Responsive:** Completely responsive UI that can be installed to your mobile home screen to provide a native feel.

:rocket: **Progressive:** Built with PWA support using `vite-plugin-pwa`.

## Themes

Built in theme engine!

Current themes:
* Default
* Night
* Black (AMOLED)

## Tech Stack

- **React 19** with TypeScript
- **Vite** for fast dev server and optimized builds
- **React Router v6** for client-side routing
- **SCSS** for styling with theme engine
- **vite-plugin-pwa** for service worker and offline support

## Build process

- Clone or download the repo
- `npm install`
- `npm run dev` to run the development server
- `npm run build` to create a production build
- `npm run preview` to preview the production build locally
