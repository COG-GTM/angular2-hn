:zap: **Fast:** Service Worker App Shell + Dynamic Content model to achieve faster load times with and without a network.

:iphone: **Responsive:** Completely responsive UI that can be installed to your mobile home screen to provide a native feel.

:rocket: **Progressive:** Built with Vite and vite-plugin-pwa for optimized PWA support.

## Tech Stack

- **React 18** with TypeScript
- **Vite** for fast development and optimized builds
- **React Router v6** for client-side routing
- **SCSS Modules** for scoped component styles
- **vite-plugin-pwa** (Workbox) for service worker and offline support

## Features

- Browse Hacker News feeds: Top, New, Show, Ask, Jobs
- View item details with recursive comment threads
- User profiles
- Settings panel with themes (Default, Night, Black/AMOLED)
- Customizable font size and list spacing
- Open links in new tab option
- PWA support with offline caching
- Lazy-loaded routes for item details and user profiles

## Offline Support

This app uses [vite-plugin-pwa](https://vite-pwa-org.netlify.app/) with Workbox to generate a service worker as part of the build step to load quickly and work offline.

## Themes

Built in theme engine!

Current themes:
* Default
* Night
* Black (AMOLED)

## Build process

- Clone or download the repo
- `npm install`
- `npm run dev` to run the development server
- `npm run build` to create a production build in `dist/`
- `npm run preview` to preview the production build locally

## Testing

- `npm test` to run tests with Vitest
