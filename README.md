## Offline Support

This app uses [Workbox](https://workboxjs.org/) to generate a service worker as part of the build step to load quickly and work offline.

## Manifest

With Chromium based browsers for Android (Chrome, Opera, etc...), the app includes a Web App Manifest (generated by `vite-plugin-pwa`) that allows you to install to your homescreen.

## Build process

The app is built with [Vite](https://vitejs.dev/). All commands run from the `react-app/` directory:

 - Clone or download the repo
 - `cd react-app`
 - `npm install`
 - `npm run dev` to start the Vite dev server (http://localhost:5173)
 - `npm run build` to type-check (`tsc -b`) and produce a production build in `react-app/dist/`
 - `npm run preview` to serve the production build locally
 - `npm run lint` to run ESLint

The service worker is generated by `vite-plugin-pwa` as part of `npm run build`, so offline/precaching behavior can be verified with `npm run preview` against the built output.

Hosting is on Firebase; `firebase.json` serves the `react-app/dist` output with SPA rewrites.
